How to use:

While in game, save, then press F10 to open a popup. Copy your save info from the first box, and paste it into the textbox in this page, then press Load Save

Apply Changes in any of the other tabs to save any changes, and come back to this tab to grab your edited save file.

To add the edited save file back into the game, press F10 and paste the edited save into the second textbox and press Accept. This will add the new save file at the bottom of the save list.

Start editing, have fun :)
